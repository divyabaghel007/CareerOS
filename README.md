# CareerOS

An AI-powered career roadmap platform that transforms a user's career aspirations into a structured learning journey.

Built as a frontend implementation for a hackathon challenge where teams were required to analyze a provided wireframe and convert it into a fully functional frontend experience while maintaining the original user flow and functionality.

---

## Challenge Statement

My task was to:

- Analyze the assigned wireframe
- Understand the intended user flow
- Transform it into a functional frontend application
- Preserve the original functionality and structure
- Avoid unnecessary redesigns

---

## Project Overview

CareerOS helps users:

1. Define their target career role
2. Select existing skills
3. Specify experience level
4. Set weekly learning commitment
5. Generate a personalized career roadmap
6. Track progress through a mission-control dashboard

The platform presents career growth as a guided mission, making long-term career planning more engaging and actionable.

---

## User Flow

### Landing Page
Users enter their desired career destination.

### Skill Assessment
Users select their current technical skills.

### Experience Selection
Users choose their current experience level.

### Time Commitment
Users specify how many hours per week they can dedicate.

### Dashboard
The platform generates a personalized roadmap with:

- Progress tracking
- Milestones
- Weekly goals
- Learning phases
- Career navigation dashboard
